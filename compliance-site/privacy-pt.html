Política de Privacidade

Última Atualização: 3 de Julho de 2026

1. Introdução

Bem-vindo ao MultiChat Integrator. Esta Política de Privacidade explica como coletamos, usamos e lidamos com suas informações quando você utiliza nosso aplicativo desktop para agregar mensagens de chat ao vivo.

2. Processamento Local do Aplicativo

O MultiChat Integrator é um aplicativo desktop local. Todo o processamento de mensagens de chat, chaves de API e configurações de usuário ocorre inteiramente na sua máquina local. Nós não operamos servidores na nuvem externos que armazenam seus dados de chat, credenciais de transmissão ou informações de espectadores.

3. Serviços de API do YouTube

O MultiChat Integrator usa os Serviços de API do YouTube para buscar mensagens de chat ao vivo de suas transmissões do YouTube. Ao usar este aplicativo e conectar sua conta do YouTube, você concorda em ficar vinculado aos Termos de Serviço do YouTube.

  • Termos de Serviço do YouTube: https://www.youtube.com/t/terms
  • Política de Privacidade do Google: Nós aderimos explicitamente à Política de Privacidade do Google. Você pode revisá-la aqui: http://www.google.com/policies/privacy

4. Quais Dados Coletamos e Por Quê

Através dos Serviços de API do YouTube, nosso aplicativo acessa as seguintes informações localmente:

  • Mensagens de Chat ao Vivo: Conteúdo em texto das mensagens enviadas durante uma transmissão ao vivo para exibi-las na interface unificada.
  • Avatares e Nomes de Usuário: Para identificar visualmente os participantes do chat na interface.

Estes dados nunca são transmitidos para nossos servidores, vendidos a terceiros ou compartilhados externamente. Eles são mantidos apenas na memória local do aplicativo durante a sessão ativa.

5. Exclusão de Dados e Revogação de Acesso

Como o MultiChat Integrator não armazena seus dados em servidores externos, não há conta remota para ser excluída. Os dados locais podem ser limpos desinstalando o aplicativo ou limpando os arquivos de armazenamento local do mesmo.

Revogando Acesso ao YouTube: Você pode revogar o acesso do MultiChat Integrator aos seus dados do YouTube a qualquer momento através da página de Configurações de Segurança do Google. Para revogar o acesso, visite:

  • Página de Permissões de Segurança do Google

6. Fale Conosco

Se você tiver alguma dúvida sobre esta Política de Privacidade, entre em contato conosco através do repositório oficial do projeto.
